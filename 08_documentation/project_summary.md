# 📊 SQL Data Warehouse & Analytics Project

## 🚀 Overview
This project builds a complete **SQL Data Warehouse solution** integrating CRM and ERP data to enable business analytics and reporting.

---

## 🏗️ Architecture
The project follows a **Medallion Architecture**:
- Bronze: Raw data ingestion
- Silver: Data cleaning & transformation
- Gold: Business-ready data model

---

## 🔧 Technologies
- SQL Server
- T-SQL
- Data Modeling (Star Schema)
- ETL Processes

---

## 📊 Key Features
- Data cleaning and standardization
- Star schema design
- Business KPI creation
- Advanced SQL analytics (window functions, cohort analysis)
- Reporting layer (Gold tables/views)

---

## 📈 Business KPIs
- Total revenue
- Total orders
- Customer segmentation
- Monthly sales trends
- Top customers and products

---

## 🎯 Outcome
This project simulates a real-world **data warehouse used for business decision-making and reporting**.
