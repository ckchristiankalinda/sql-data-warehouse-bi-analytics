# 📘 Data Dictionary - SQL Data Warehouse Project

## 🧾 Fact Table: fact_sales

| Column | Description |
|--------|------------|
| order_number | Unique identifier of each order |
| customer_id | ID of customer making purchase |
| product_id | ID of product sold |
| order_date | Date of order |
| sales_amount | Total revenue generated by the order |
| quantity | Number of items purchased |

---

## 👤 Dimension Table: dim_customers

| Column | Description |
|--------|------------|
| customer_id | Unique customer identifier |
| first_name | Customer first name |
| last_name | Customer last name |
| gender | Customer gender |
| marital_status | Marital status |
| create_date | Account creation date |

---

## 📦 Dimension Table: dim_products

| Column | Description |
|--------|------------|
| product_id | Unique product identifier |
| product_name | Name of product |
| category | Product category |
| price | Unit price |

---

## 🎯 Purpose
This data warehouse is designed for:
- Sales analysis
- Customer segmentation
- Product performance tracking
- Business KPI reporting
